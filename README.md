# VertiCar
Balancing UGV base


## Description 

Primary objective is to build remotely controlled, vertically balancing driving base for future UGV development.

This is a presonal portfolio projekt where the goal is to gain knowledge and experience in building real life UGV solutions. 



## Design and Functionality objectives:
- oop code for embedded systems
- implementation of FreeRTOS
- reliable controll algorithm 
- reliable communication 
- implemented safety management
